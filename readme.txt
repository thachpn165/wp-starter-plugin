A WordPress plugin boilerplate.

== Description ==

My Plugin is a boilerplate for creating WordPress plugins with modern development practices.

**Features:**

* PSR-4 autoloading with Composer
* Modern PHP (7.4+)
* Admin settings page
* Docker development environment
* Build scripts for deployment
* Code quality tools (PHPCS, PHPUnit)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/my-plugin`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to My Plugin in the admin menu to configure
